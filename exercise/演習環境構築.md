## 環境構築関連
### 2025-03-08
#### jestのインストール
```text
npm install --save-dev jest@latest ts-jest@latest @types/jest@latest
```
#### node_modules を削除して再インストール
```text
rm -rf node_modules package-lock.json
npm install
```
